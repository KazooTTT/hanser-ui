# hanser弹幕组UI库（Vue版）

1. 命名规范

    1.1 所有组件名必须以 **`h-`** 开头

    1.2 css规范

    - a 不使用id
    - b 标签统一使用 `小写` 加 `-` (短横线)
    - c 类名由以下元素构成
       - `小写字母`
       - `-` (短横线)
       - `_` (下划线)
       - `数字`
    - d 使用 **BEM** 命名规范: `块__元素--修饰符`<br>例:<br>.footer__logo--dark (黑暗模式的底部LOGO)<br>.user-rank__avatar--first (用户棒排名第一的用户头像)<br>

2. 创建新组件

   2.1 运行 **`npm run component 组件名 注释`** 创建组件
